# Real Estate App

تطبيق إدارة العقارات المتكامل، يتيح للمستخدمين إدارة العقارات، العقود، الدفعات، الشكاوى، والدردشة بين الأطراف. المشروع يتكون من جزئين رئيسيين:

- **backend/**: خادم Node.js باستخدام Express و MongoDB.
- **flutter_application_1/**: تطبيق Flutter متعدد المنصات (Android, iOS, Web, Windows, macOS, Linux).

---

## المتطلبات

### Backend
- Node.js 18 أو أحدث
- MongoDB (محلي أو عبر Atlas)

### Flutter App
- Flutter SDK 3.9 أو أحدث
- Android Studio أو VS Code

---

## طريقة التشغيل

### 1. تشغيل الـ Backend

```sh
cd backend
npm install
# ضع بيانات الاتصال بقاعدة البيانات في ملف .env
npm start
```

### 2. تشغيل تطبيق Flutter

```sh
cd flutter_application_1
flutter pub get
flutter run
```

## الميزات الرئيسية

- **المستخدمون**: تسجيل/دخول، إدارة الصلاحيات (مدير، مستأجر، مالك).
- **العقارات**: إضافة، تعديل، حذف، بحث جغرافي.
- **العقود**: إنشاء، عرض، تعديل، حذف، ربط العقود بالمستخدمين والعقارات.
- **الدفعات**: إدارة الدفعات المرتبطة بالعقود.
- **الشكاوى**: تقديم ومتابعة الشكاوى.
- **الدردشة**: رسائل بين المستخدمين.
